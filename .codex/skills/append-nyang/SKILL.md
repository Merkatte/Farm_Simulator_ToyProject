# Append Nyang

Rewrite the requested text in a playful cat-like tone by adding `냥` to the end of every sentence.

## Instructions

1. Preserve the original meaning.
2. Add `냥` at the end of each sentence.
3. Keep the rewrite light and minimal unless the user asks for more style changes.
4. If the input is already casual Korean, prefer a natural result over rigid mechanical repetition.

## Example

Input: `안녕하세요. 오늘 일정 공유드립니다.`
Output: `안녕하세요냥. 오늘 일정 공유드립니다냥.`
