# 협업 에이전트 제한 사항
- `AGENT.md` 파일과 `.codex` 폴더는 절대 확인하거나 열어보지 않는다.
- 해당 파일 및 폴더는 타 회사 AI 에이전트인 Codex 전용이며, Claude는 접근하지 않는다.

# AI_CONTEXT_Public 공용 문서

Claude와 Codex 두 AI 에이전트가 공동으로 참조하는 문서 폴더다. 코드 작업 전 관련 파일을 반드시 확인하고, 작업 후에는 `update_context` 스킬로 내용을 갱신한다.

| 파일 | 역할 | 참조 시점 |
|------|------|-----------|
| `Architecture.md` | 프로젝트 전체 구조, 구현된 시스템(폴더 구조, 스크립트 목록, 의존 관계, Scene 조립 가이드, 미구현 예정 목록) | 새 스크립트 추가 / 기존 시스템 수정 / 의존 관계 파악 시 |
| `code_convention.md` | C# 코드 작성 기준 (K&R 괄호, 네이밍 규칙, 주석 규칙, 데이터 외부화, SOLID/Interface DI, Singleton 제한, 폴더 구조, 커밋 체크리스트) | 코드 작성 전 반드시 확인 |
| `log.md` | Claude·Codex 공용 작업 이력 (날짜별 변경 내역) | 과거 맥락 파악 시, 작업 완료 후 기록 추가 시 |
